# flask-basic

​		flask基础的学习

​		demo中包含了部分知识点

## 项目内容

​		用户的登录登出

​		用户注册

​		验证码

​		发布问答

​		评论

​		搜索功能

### 项目额外导入的包

​	在Flask框架的基础上需要另外下载的包:

​			functools：用于装饰器的实现

​			flask_sqlalchemy：替代SQL语句的书写

​			flask_mail：使用邮箱发送邮件（验证码功能）

​			werkzeug.security：加密

​			flask_migrate：建立对象关系映射，更新数据库的表

​			wtforms.validators：用于表单验证
